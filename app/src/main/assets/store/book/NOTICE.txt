# Embedded Rust Book chapters

The `*.md` files in this directory are chapters of *The Rust Programming
Language* (a.k.a. "the Rust Book"), vendored from <https://github.com/rust-lang/book>
(`main` branch) so the Book reader works fully offline — no CDN, no install-time
fetch. Only the chapters referenced by the exercises' `book_refs` are included.
